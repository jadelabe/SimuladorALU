# ${Simulador de Unidad Aritmetico Logica}
Practica 4 de Arquitectura de Computadores en UniLeon, creacion de simulador de Unidad Aritmetico Logica.

Simulacion de operaciones basicas de una ALU 
Suma 			---- Implementada con menores fallos en el redondeo
Multiplicacion 	---- Implementada
Division 		---- WIP

El Programa ha sido compilado y testeado bajo Windows 10 x64

## Compilacion
Compilar desde Qt cargandolo con el archivo .pro

## Usage
Introducir los numeros deseados en los campos correspondientes y hacer click sobre la operacion a realizar
El separador de decimales es un punto "." 

Operador1 + Operador2 = Resultado
Operador1 * Operador2 = Resultado
Operador1 / Operador2 = Resultado
